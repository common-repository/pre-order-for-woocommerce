== Description ==

**Woocommerce Pre Order** using to setup purchase product before item available in inventory.

Waiting for a new product (or the return of a popular one) is a perfect time to attract new customers or engage existing ones. However, far too many businesses waste this once-in-a-lifetime opportunity, causing clients to lose interest. In the days leading up to a product launch, engagement is crucial, which is why we designed a plugin that covers all bases. Allow clients to **pre order plugin woocommerce** so they can obtain their new item in a timely manner.

> **Features**
>
> * Easy Setup
> * Customize Button Title
> * Customize Availabe Text
> * Setup Pre Order Date
> * Get Notify admin if a Pre-Order product is purchased
> * Support Single Product as well Variation Product

**Woocommerce Pre Order Free** can be set up in your WooCommerce store so that buyers can order things before they are released. When you're ready, you may release and fulfil orders automatically or manually, and the extension will take care of the rest!
